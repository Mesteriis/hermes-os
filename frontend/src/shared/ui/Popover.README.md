# Popover

## Description
Small interactive contextual overlay.

## When to use
Use for local controls, compact settings and contextual details.

## When not to use
Do not use for modal confirmation or large persistent panels.

## Accessibility
Built on Reka Popover and exposes trigger/content semantics.

## Keyboard
Escape closes the popover. Trigger and content keep normal focus behavior.

## Examples
`<Popover><template #trigger>...</template>...</Popover>`

## Anti Patterns
Do not put required page content only in a popover.
