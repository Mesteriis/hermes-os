# Statistic

## Description
Large numeric summary card.

## When to use
Use for high-level counts or review summaries.

## When not to use
Do not use for dense multi-row data.

## Accessibility
Keeps label and value in the same article.

## Keyboard
No custom keyboard behavior is added.

## Examples
`<Statistic label="Signals" :value="12" />`

## Anti Patterns
Do not compute business metrics inside the component.
