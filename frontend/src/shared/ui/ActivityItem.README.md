# ActivityItem

## Description
Compact activity-feed row.

## When to use
Use for local UI summaries of recent actions.

## When not to use
Do not use for structured tables or editable records.

## Accessibility
Renders a readable article with decorative icon.

## Keyboard
The component does not create interactive behavior.

## Examples
`<ActivityItem title="Context refreshed" meta="now" />`

## Anti Patterns
Do not trigger side effects from display rows.
