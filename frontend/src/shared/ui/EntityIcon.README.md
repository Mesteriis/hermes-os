# EntityIcon

## Description
Icon primitive for durable Hermes entity categories.

## When to use
Use in lists, cards and metadata rows that need entity type markers.

## When not to use
Do not fetch or resolve entity records here.

## Accessibility
The wrapper exposes an accessible image label.

## Keyboard
No keyboard behavior.

## Examples
`<EntityIcon entity="project" label="Project" />`

## Anti Patterns
Do not create domain-specific entity behavior in this primitive.
