# KeyValue

## Description
Single labeled value primitive.

## When to use
Use inside description lists, property grids or small inspectors.

## When not to use
Do not use for standalone prose.

## Accessibility
Renders `dt` and `dd` pairs for assistive technology.

## Keyboard
No custom keyboard behavior is added.

## Examples
`<KeyValue label="Source" value="Local" />`

## Anti Patterns
Do not encode validation or business truth inside value formatting.
