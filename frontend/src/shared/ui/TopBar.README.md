# TopBar

## Description
Top application or panel bar with start, main and end regions.

## When to use
Use for local shell headers, workspace headers and panel titles.

## When not to use
Do not use as a marketing hero or domain-specific header.

## Accessibility
Defaults to `header`; title and description are visible text, not hidden metadata.

## Keyboard
TopBar has no keyboard behavior.

## Examples
Use `#start` for icons and `#end` for actions.

## Anti Patterns
Do not place large body content in a TopBar.
