# Counter

## Description
Inline count or ratio badge.

## When to use
Use for local counts near tabs, filters or labels.

## When not to use
Do not use for long text or primary actions.

## Accessibility
Optional label becomes the accessible name.

## Keyboard
No custom keyboard behavior is added.

## Examples
`<Counter :value="3" label="pending" />`

## Anti Patterns
Do not use counters as buttons without an explicit button wrapper.
