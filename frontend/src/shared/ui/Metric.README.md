# Metric

## Description
Compact labeled metric with unit and delta.

## When to use
Use for dashboards, inspectors and repeated metric rows.

## When not to use
Do not use for prose or unstructured status text.

## Accessibility
Label, value, unit and delta are rendered as text.

## Keyboard
No custom keyboard behavior is added.

## Examples
`<Metric label="Confidence" value="82" unit="%" />`

## Anti Patterns
Do not hide calculations or scoring policy in this primitive.
