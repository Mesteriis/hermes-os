# Dialog

## Description
Modal content surface for focused UI tasks.

## When to use
Use for temporary forms, confirmations that are not destructive, and focused local workflows.

## When not to use
Do not use for page layout, navigation or persistent panels.

## Accessibility
Built on Reka Dialog with modal semantics, overlay, title and description support.

## Keyboard
Escape closes the dialog. Tab stays within the dialog while it is open.

## Examples
`<Dialog title="Create item">...</Dialog>`

## Anti Patterns
Do not hide long domain workflows inside a dialog.
