# NoDataState

## Description
Specific no-data state for valid surfaces without local data.

## When to use
Use when there is no dataset to render.

## When not to use
Do not use for search filtering misses.

## Accessibility
Provides readable heading and description.

## Keyboard
No custom keyboard behavior is added.

## Examples
`<NoDataState />`

## Anti Patterns
Do not claim remote systems have no data from this component.
