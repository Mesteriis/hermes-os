# ScrollArea

## Description
Tokenized scroll container using the shared Hermes/Reka boundary.

## When to use
Use for local scroll regions inside panels, lists and component previews.

## When not to use
Do not use for page-level browser scrolling.

## Accessibility
Content semantics are preserved inside the viewport.

## Keyboard
Native browser scroll behavior applies.

## Examples
`<ScrollArea size="md">...</ScrollArea>`

## Anti Patterns
Do not hide focusable content in tiny scroll containers.
