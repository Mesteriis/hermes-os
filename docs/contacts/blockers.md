# Contacts — Архитектурные блокеры

**Блокеров нет.** Модуль контактов реализован полностью.

Все функции спецификации (§18-19) покрыты:
- Авто-создание из писем
- Identity resolution (merge/split)
- Communication fingerprint (эвристики + LLM)
- Обогащённый профиль (язык, тон, trust score, время ответа, избранное, заметки)
- Связь с проектами
- Поиск

Единственное потенциальное улучшение — LLM-based fingerprint даёт более точные результаты при наличии Ollama, но эвристики работают и без него.
